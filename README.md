# Assignment-3---Rock-Paper-Scissors
CS151 Assignment 3 - Rock, Paper, Scissors

## Prerequisites

- Install a **Java JDK** (Java 17+ recommended).
- Verify it’s available:

```bash
java -version
javac -version
```

## Project layout

- **Source**: `src/`
- **Scripts**: `scripts/`
- **Build output (generated)**: `out/` (ignored by git)

## Compile

```bash
./scripts/compile.sh
```

## Run the game

```bash
./scripts/run.sh
```

## Run tests

The tests are self-contained Java programs with `main()` methods under `src/test/`.

```bash
./scripts/test.sh
```
